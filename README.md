# Exploratory Data Analysis (EDA) в pandas

Этот репозиторий содержит проекты по Exploratory Data Analysis (EDA), выполненные на Python с использованием pandas и сопутствующих библиотек.

**Цель репозитория** — продемонстрировать системный, воспроизводимый и ориентированный на бизнес-подход к исследовательскому анализу данных: от первичного знакомства с датасетом до проверки корректности данных, выявления аномалий, подготовки признаков и формулирования аналитических выводов.
